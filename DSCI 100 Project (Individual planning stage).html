Methods and Plan

To answer my question about whether player characteristics and activity features can predict newsletter subscription, I will use a supervised classification method. My response variable is subscribe, which is a categorical outcome with two possible values, so a classification model is appropriate. For this project, I plan to use logistic regression as my primary method because it is simple, interpretable, and works well when the goal is to understand how different features influence the probability of an event. Logistic regression will allow me to examine how variables such as total hours played, number of sessions, and other activity-related features relate to the likelihood of subscribing. This method produces coefficients that show the direction and strength of relationships, which helps when interpreting player behaviour.

Logistic regression assumes that each observation in the dataset is independent from the others. It also assumes that the predictors have a linear relationship with the log odds of the outcome. Another assumption is that the explanatory variables should not be too strongly correlated with each other. These assumptions are reasonable for this dataset, but I will still check the correlations between variables to make sure that none of them overlap too much or carry the exact same information.

A possible limitation of this method is that the dataset is imbalanced because there are more subscribed players than unsubscribed players. This imbalance can influence how accurate the predictions are, so I may need to adjust the classification threshold or use weighting when training the model. Logistic regression also might not capture complex or non-linear relationships between variables. If the model does not perform well, I will compare it with a decision tree or a random forest to see if these methods can identify more detailed patterns in the data.

For data processing, I will split the data into two sets. I will use eighty percent for training and twenty percent for testing. This will allow me to evaluate the model on unseen data. I may also use cross-validation on the training set to select the best model and tuning strategy.
